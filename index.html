Zero Sum Game Of Life

Zero Sum Game Of Life is a living grid of integers. When a cell's value exceeds a threshold, it spills into other cells. The result is a fascinating cellular automaton which can produce beautiful fractals. Check it out in action here.
Human Paintbrush

The Human Paintbrush Project is a collaborative real-time art game for Android. Players can download the app and use it to "become" a paintbrush. Asteroids

This is a clone of the classic Asteroids arcade game. Created with HTML 5 canvas, it features a seemless torus topology (wrapping), and redundant rendering for consistency at viewport edges. It also features optional mouse controls. Try it out and see what score you can get, or feel free to check out the source code on github.
Guitar Tuner

Tune your guitar strings with this app. This app will use the microphone to detect the pitch of your guitar strings. It will automatically detect which string you are tuning, and provides a gauge to easily tune with precision. Come check it out. Or, fork the code if you want to contribute.
Pretty Lashes of Brine

In 2013, to test the boundaries of what is possible in our digital world, I challenged myself by building a website from a remote rural location in Central America. I was able to acquire a domain name, launch an Amazon instance, configure a full LAMP stack with Wordpress, and put up some simple content, in one evening after dinner. This was all accomplished via a satelite internet connection running off a generator in an area that lacks even a basic power grid. Come check it out.
French Press Timer

Make a nice french press coffee with convenience. This app will set two timers, and will let you know when it is time to mix the bloom, and press the coffee. You can customize the timer durations and the phrase used to let you know it's time to mix or press. Come check it out.

Chess

This is a command line chess game written in Ruby. It features a board made with unicode box characters. Take turns against an opponent by typing out your moves. Check it out on github!
